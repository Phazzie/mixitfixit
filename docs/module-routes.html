(async, inner) /api/ai-summarize(req, res) → {Promise.<void>}

Handles POST requests to /api/ai-summarize. Takes user submitted statements and returns an AI generated summary. Responds with status 400 for bad inputs, 500 for AI errors, 403 for permission errors and 200 for successful requests.
