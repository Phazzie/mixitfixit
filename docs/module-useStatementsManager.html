Module: useStatementsManager

Hook to manage statements. This hook provides functions to retrieve and add statements, interacting with local storage.

(inner) addStatement(code, user, text, setError)

Adds a new statement to the local storage. This function validates the input, retrieves existing statements, adds the new statement, and then updates local storage. If there is an error in the validation, or in the try...catch, it will call the setError function.

Parameters:
Name Type Description
code string

The code associated with the statement.
user string

The user who made the statement.
text string

The text of the statement.
