This module provides the useSummarizeStatements hook, which manages the process of summarizing statements by sending them to the server's AI summarization API. It also manages the state of the summarization request, including the AI's response and any errors that occur during the process.

Methods

(async, inner) handleSummarizeClick(code)

This asynchronous function initiates the summarization process. It clears previous AI response and API error. It retrieves the most recent statements for "Participant 1" and "Participant 2", then sends the statements to the server for summarization. If successful, it will set the aiResponse. If there was an error, it will set the apiError.

(inner) useSummarizeStatements(getMostRecentStatement, post, setError) → {object}

React hook for handling the summarization of statements. It sends the most recent statements to an AI summarization API and manages the state of the AI's response and any potential errors.

Name Type Description
getMostRecentStatement function

A function that retrieves the most recent statement for a given participant and code. This function is expected to accept two parameters: the code and the participant name. This function will also receive the setError function as a parameter.

The post function that will be used to make the api calls.

The function to use to set an error.
